# Canyon War MOBA

A browser-based MOBA game inspired by Honor of Kings.
**All skins are FREE and earned through gameplay!**

## How to Play

1. Double-click `start.bat` to start the server
2. Open `http://localhost:3456` in your browser
3. Or share your LAN IP with friends: `http://YOUR_IP:3456`

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Aim |
| Left Click | Attack |
| Q | Skill 1 - Sword Slash (6s) |
| W | Skill 2 - Whirlwind (8s) |
| E | Skill 3 - Dash (10s) |
| R | Ultimate - Earth Shatter (40s) |
| B | Open Skin Shop |
| Space | Center Camera |
